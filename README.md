# Fragmentation-Reassembly-of-IP-packets

For Linux G++:

-->g++ server.cpp -lpthread -o server

--> ./server

--> g++ client.cpp -o client

--> ./client

>run server and client in seperate terminals.
>change the *ip to the local ip of the machine you are running.
